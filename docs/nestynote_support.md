# NestyNote Support

[← Back to App Launchpad]({{ '/' | relative_url }})

We’re here to help you get the most from NestyNote. Use this page to request assistance, report issues, or share feedback.

## Contact the team

- **Email:** [moocbear@gmail.com](mailto:moocbear@gmail.com)
- **Support hours:** Monday–Friday, 9:00–17:00 EST
- **Response target:** within 2 business days

When you reach out, please include:

1. Your NestyNote app version (Settings → bottom of page).
2. Device model and OS version.
3. A clear description of the issue, reproduction steps, and screenshots if available.

## Common topics

- **Account or sign-in issues:** Include the email/nickname you used to register and any error text shown.
- **Data or sync questions:** Tell us which view (Daily, Graph, Event Detail) and what looks incorrect.
- **Feature suggestions:** Describe how the change would improve your workflow so we can prioritize it.

## Service status

All NestyNote services are operating normally. We’ll post maintenance or outage updates here if that changes.

## Escalation

For urgent issues, add “URGENT” to the email subject. We monitor that inbox closely and will respond as quickly as possible.
